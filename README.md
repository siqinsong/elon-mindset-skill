## What Is This?

**elon-mindset** is a Claude Cowork skill that channels Elon Musk's philosophy, decision frameworks, and execution principles. Based on a comprehensive extraction of *The Book of Elon* (bilingual Chinese-English edition), it lets you:

- Apply **First-Principles Thinking** to any problem ("Physics is law. Everything else is a recommendation.")
- Run decisions through **The Algorithm** — Musk's 5-step engineering process
- Think at **civilizational scale** (internet → energy → space → AI → genetics)
- Challenge "impossible" constraints and escape analogy-based reasoning
- Evaluate startups, life choices, and strategies with Musk's utility equation: `Utility = people helped × degree of help`

## Core Frameworks Inside

### 1. 追求使命 — Pursue Purpose
- Utility equation as life compass
- 5 civilizational mission areas
- "Create more value than you consume"

### 2. 像物理学家思考 — Think Like a Physicist
- First-principles 4-step deconstruction
- "I always assume we're losing" — radical honesty
- Escape analogy reasoning, build from axioms

### 3. 超硬核工作 — Ultra Hardcore Execution
- 80–100 hrs/week commitment philosophy
- Accept fear, accept low odds (SpaceX <10%)
- "Eat glass and stare into the abyss"

### 4. 建立伟大公司 — Build Great Companies
- **The Algorithm**: Question → Delete → Simplify → Accelerate → Automate
- Team = Special Forces (10 A-players > 100 average)
- Remove org boundaries; manufacturing as moat

### 5. 星际视野 — On Behalf of Humanity
- Companies as vehicles for civilizational philanthropy
- Multiplanetary species rationale
- Existential risk framework

## Installation

### Option A — Install .skill file directly (Recommended)

1. Download [`elon-mindset.skill`](./elon-mindset.skill)
2. Open **Claude Cowork** desktop app
3. Go to **Plugins → Install from file**
4. Select the downloaded `.skill` file
5. Done! The skill is now available in your session.

### Option B — Manual install

1. Download or clone this repo
2. Copy the `elon-mindset/` folder into your Cowork skills directory:
   ```
   ~/.claude/skills/elon-mindset/
   ```
3. Restart Cowork
